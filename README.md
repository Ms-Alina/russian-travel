# Проект: Путешествие по России

Адаптивный одностраничный сайт, посвещенный живописным местам России.

**В проекте использованы:**
* HTML разметка
* БЭМ структура
* CSS свойства
* Flexbox
* Grid
* Резиновая верстка
* Адаптивная верстка
